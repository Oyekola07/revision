Comparison Operator

These are use in logical operations to compare numbers, strings and expressions or statements which the outcome can either be a true or false.

below are the examples of comparison operator, description and examples:

1.   Equal to (==): This returns a value of true or false after comparing the value of 2 variables e.g

Ade == Sunday

2.   Equal value and equal type(===): This returns a value of true if both the value and the datatype of variable is the same else returns a false. e.g

Jessica===Augustine

3.   Not Equal to (!=): This returns a value of true (boolean value) after checking if it two operand are not equal else returns false value. e.g

140 != $140

4.   Not Equal value or Not equal type (!==): This returns a value of true if one of either value or datatype of the operands is the same else return a false value. e.g

100 !== 1000

5.   Greater than(>): The greater than operator returns true if the left operand is greater than the right operand, and false otherwise. e.g

100 > 400

6.   Less than (<): The less than operator returns true if the left operand is less than the right operand, and false otherwise e.g

340 < 400

7.   Greater than or Equal to (>=): The greater than operator or equal to returns true if the left operand is greater than or equal to the the right operand, and false otherwise e.g

1001 >= 1001

8.   Less than or Equal to (<=): The less than or equal to operator returns true if the left operand is less than or equal to the right operand, and false otherwise e.g

1001 <= 1001
