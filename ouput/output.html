Output

There are various ways to display or output content is javascript, even though they all display content but are different from each other.

1.   window.alert ("statement"): This create an output in form of pop up

2.   document.write (""): As the name implies, this simply write an output to display

3.   console.log (""): This display an output in the browser console.

4.   DOM document Object Model document.getElementById().innerHTML= This display an output on the browser.

Method 2 and 3 are used for testing and are not recommended to be used within project
